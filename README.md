# OpenCode Setup

> Analyse tes projets et configure automatiquement OpenCode — MCP servers, outils, skills, agents.

## Installation

```bash
curl -fsSL https://raw.githubusercontent.com/Sunderrrr/opencode-setup/main/install.sh | bash
```

## Utilisation

Lance `/setup` dans OpenCode pour analyser ton projet et recevoir des recommandations.

Ou en CLI :

```bash
opencode-setup scan /chemin/vers/mon/projet
```

## Dépendances

- Python 3.10+
- OpenCode 1.17+
